## [Download the Mac App](https://s3-us-west-2.amazonaws.com/assets.dashboard/AWS+Profile+Manager.app.zip)

Or install locally

1. Clone down this repo
2. `npm install` dependancies
3. `npm start` && `npm run dev` in two terminal windows

Or package locally

`npm run package`
